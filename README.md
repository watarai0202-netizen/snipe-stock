1. プロジェクト名
Pre-Market Sniper (5MA & RVOL Strategy)

2. 戦略概要（コア・ロジック）
このシステムは、**「前日に資金が入り、壁を抜けた最強銘柄が、翌朝一瞬だけ見せる押し目」**を狙撃するためのものです。

エントリータイミング: 9:15 - 9:30（寄り付きのパニック一巡後）。

ターゲット: 日足5MA（5日移動平均線）付近での反発。

選定基準:RVOL（相対出来高）: 1.2倍 〜 1.3倍（過熱しすぎない熱量）。

ブレイクアウト: 直近5〜10日の高値を更新していること。

需給: 松井証券の当日推計データに基づき、信用買残のしこりが少ないこと。

3. 失敗回避（フィルター）
「寄り底偽装の下げ」や「レンジ膠着」を避けるための判定基準を明記します。

先物V字判定: 8:30時点の戻り率が60%以上なら実行。

先物L字判定: 戻りが鈍い場合は、指値を下げるか注文をキャンセル。

GU制限: 前日比+3%以上のギャップアップは「寄り天」リスクのため除外。

4. 運用フロー

日々のルーチンを記載します。

前夜: スクリーニングによる候補選定（RVOL & ブレイク）。

当日08:40: 松井証券の需給データを手入力。

当日08:50: 先物トレンドによる最終指値の算出と発注。

当日10:00: 30分ディレイデータによる答え合わせ（自動ログ記録）。
